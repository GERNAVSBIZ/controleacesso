Gerenciamento e Backup de Dados

Os dados são armazenados localmente e de forma segura no seu navegador utilizando IndexedDB. Use as ferramentas abaixo para fazer o backup de segurança e importar dados ao trocar de computador.

Exportar Backup (JSON)

Exporta toda a base de dados em formato legível por máquina para restauração futura.

Exportar Planilha (CSV)

Exporta os acessos em formato CSV (separado por ponto e vírgula), perfeito para abrir no Excel.

Importar Backup (JSON)

Carrega um arquivo JSON gerado anteriormente para restaurar a lista de visitas.

Zona de Risco

Esta ação apagará permanentemente todos os registros do sistema do seu navegador. Certifique-se de ter um backup antes de prosseguir.
